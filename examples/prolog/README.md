The `prolog.pf` file supplies an implementation of an ISO-compliant implementation of Prolog.

Specifically, it wraps around the Ichiban implementation by Yutaka Ichibangase, found here.

https://github.com/ichiban/prolog
https://github.com/ichiban/prolog/wiki

The `service.pf` file is a little demonstration of how one might use it to make a specialized database app, in this case one dedicated to the traditional question of which Greeks are mortal.
